Girls Get Coding! Senior Project

Laravel // PHP // Sass

Girls Get Coding is a way to introduce adolescent girls to STEM related topics, primarily programming. This project is designed as a means to create and foster a desire for Computer Science in a young girl's life.

Problem

Currently there is a shortage of female involvement in Computer Science, this problem starts at an early age and is related to a stigma that computers and games are geared more towards males. This project is aiming to challenge these stereotypes and create an influx of female involvement in Computer Science now and into the future.

Desired Impact

The desired impact of this project is to introduce programming to girls at a young age within the Missoula community. By presenting the concepts of programming in simple, approachable ways in personal 1-on-1 tutoring sessions, the development team hopes that young Missoula girls will be more inclined to pursue programming and in turn be open to making a career out of Computer Science later in their lives. The end goal of this project for is that this generated interest in technology may lead eventually to a smaller, if non-existent, gender gap in STEM career fields.

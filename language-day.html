Language Day Form - University of Montana

Laravel // HTML // CSS // JS

The web application was requested and built to automate registration process and class scheduling for the Language Day at the University of Motnana. Language Day is a program ran by the English department at the University of Montana.

As one of my first projects as part of the SFD team, I was given the task to develop the web application for Language Day. The application essentialy works to give the admin a more controlled overview of the event with class and teacher rosters, class schedule and payments record.

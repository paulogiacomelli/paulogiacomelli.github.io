Language Day Form - University of Montana

Laravel // HTML // CSS // JS

The web application was requested and built to automate registration process and class scheduling for the Language Day at the University of Montana. Language Day is a program ran by the English department at the University of Montana.

As one of my first projects as part of the SFD team, I was given the task to develop the web application for Language Day. The application essentially works to give the admin a more controlled overview of the event with class and teacher rosters, class schedule and payments record.

Professors invited to the event can enter their email in the language day website, who will then receive an email with a link to enter some profile information, what students that are coming and which class they would like to attend. Prior to this application, the employees at SFD had to manually change code to show that a class was full for example. And the admin running the event had to manually keep track of student/teacher roster as well as who actually had paid for the event.
